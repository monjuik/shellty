⚡ Speed Proven in the Lab — and Ready for Production

Shellty runs fast under banking-grade workloads.

In our lab, the reference setup (100 K synthetic clients, 2-core / 4 GB VM) consistently delivered sub-millisecond response times for read scenarios and low-millisecond latency for event writes.

Laboratory benchmark

  • Read statements: 0.8 ms mean / 1.2 ms p90
  • Create events: 1.1 ms mean / 1.6 ms p90

Real-world test scenario

  • Read statements (500 RPS): ≈ 1.4 ms mean / 2.1 ms p90
  • Event processing (50 RPS with synchronous WAL writes in addition to read statements): ≈ 2.7 ms mean / 4.1 ms p90

All tests were executed on a compact 2-core / 4 GB virtual machine that still had spare CPU and memory capacity. In typical deployments with database-backed storage (using PostgreSQL), response times remain comfortably within the low-millisecond range.

Result:
Customer lookups stay instant, event streams stay reliable — even under continuous load. Engineered to stay very fast on minimal hardware.
